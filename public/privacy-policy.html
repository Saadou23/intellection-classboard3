🔐 Politique de Confidentialité

INTELLECTION ClassBoard - Application Mobile

1. Introduction

INTELLECTION ("nous", "notre" ou "nos") exploite l'application mobile INTELLECTION ClassBoard (l'«application»). Cette page vous informe de nos politiques concernant la collecte, l'utilisation et la divulgation des données personnelles lorsque vous utilisez notre application.

Nous nous engageons à protéger votre vie privée. Si vous avez des questions supplémentaires concernant notre politique de confidentialité, n'hésitez pas à nous contacter à l'adresse privacy@intellection.ma

2. Définitions

  • Données personnelles : Toute information qui peut vous identifier directement ou indirectement
  • Application : INTELLECTION ClassBoard, application mobile disponible sur iOS et Android
  • Utilisateur : Toute personne utilisant l'application (étudiants, professeurs, administrateurs)
  • Compte : Profil d'utilisateur créé dans l'application avec identifiants d'authentification

3. Données Collectées

Nous collectons les types de données personnelles suivants :

3.1 Données d'Authentification

  • Nom d'utilisateur / Email
  • Mot de passe chiffré
  • Identifiant unique d'utilisateur
  • Niveau d'étudiant ou rôle (professeur/admin)

3.2 Données d'Activité

  • Sessions de cours suivies
  • Présences enregistrées (scans QR code)
  • Heures de check-in/check-out
  • Horaires et plannings consultés

3.3 Données Techniques

  • Adresse IP
  • Type d'appareil et système d'exploitation
  • Identifiant d'appareil
  • Jeton de notification push (Expo Push Token)
  • Logs d'application

3.4 Données de Localisation

  • Centre/Filiale sélectionné (Hay Salam, Doukkali, Saada)
  • Localisation approximative si l'utilisateur l'autorise

4. Utilisation des Données

Nous utilisons vos données personnelles pour :

  • Créer et gérer votre compte utilisateur
  • Enregistrer et tracker votre présence aux cours
  • Générer des rapports de fréquentation et de discipline
  • Envoyer des notifications concernant les cours et annonces
  • Améliorer et personnaliser votre expérience dans l'application
  • Respecter les obligations légales et réglementaires
  • Analyser les tendances d'utilisation et améliorer nos services
  • Détecter et prévenir les fraudes ou abus

5. Base Légale du Traitement

Nous traitons vos données personnelles sur les bases légales suivantes :

  • Exécution du contrat : Données nécessaires pour fournir l'application (authentification, suivi)
  • Obligation légale : Conservation des données de présence requise par la loi
  • Intérêt légitime : Sécurité, prévention des fraudes, amélioration des services
  • Consentement : Pour les notifications push et analytics (consentement explicite demandé)

6. Partage des Données

Nous ne partageons pas vos données personnelles avec des tiers, sauf dans les cas suivants :

  • Administrateurs de l'établissement : Pour les rapports de présence et disciplinaires
  • Professeurs : Données limitées aux étudiants de leurs cours
  • Prestataires de services : Hébergement cloud (Firebase), services d'analyse
  • Obligation légale : Si la loi nous y oblige
⚠️ Important : Nous ne vendons jamais vos données personnelles à des tiers à des fins commerciales.

7. Sécurité des Données

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données personnelles :

  • Chiffrement en transit (HTTPS/TLS)
  • Mots de passe chiffrés avec hachage sécurisé
  • Authentification par jeton Firebase
  • Accès limité aux données (contrôle d'accès basé sur les rôles)
  • Sauvegarde régulière des données

Cependant, aucune transmission de données sur Internet n'est 100% sécurisée. Nous ne pouvons garantir une sécurité absolue.

8. Durée de Conservation des Données

Nous conservons vos données personnelles pendant la durée nécessaire pour :

  • Données de cours actifs : Pendant toute votre inscription
  • Données de présence : 3 à 5 ans (conformité légale)
  • Données de log : 30 jours
  • Après suppression de compte : 90 jours (pour récupération d'urgence)

9. Vos Droits

Conformément à la loi, vous avez les droits suivants concernant vos données personnelles :

  • Droit d'accès : Accéder à vos données personnelles stockées
  • Droit de rectification : Corriger les données inexactes
  • Droit à l'effacement : Demander la suppression de vos données
  • Droit à la limitation : Limiter le traitement de vos données
  • Droit à la portabilité : Recevoir vos données dans un format structuré
  • Droit de rétractation : Retirer votre consentement pour les notifications

Pour exercer ces droits, veuillez nous contacter à privacy@intellection.ma avec votre demande détaillée.

10. Notifications Push

Nous envoyons des notifications push pour vous informer de :

  • Annonces concernant les cours
  • Changements de statut de session
  • Rappels de devoirs
  • Messages administratifs importants

Vous pouvez désactiver les notifications push à tout moment dans les paramètres de votre appareil.

11. Enfants

INTELLECTION ClassBoard n'est pas destiné aux personnes de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles auprès d'enfants. Si nous apprenons que nous avons collecté des données d'un enfant, nous les supprimerons immédiatement.

12. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité de temps à autre. Nous vous notifierons par email ou par notification in-app de tout changement important. Votre utilisation continue de l'application après les modifications signifie votre acceptation des changements.

  • Politique de Google Play Store
